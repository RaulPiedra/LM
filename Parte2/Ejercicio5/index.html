En SEO, el término "long tail" (literalmente "cola larga") se usa para referirnos a los términos de búsqueda muy específicos (generalmente con una frase de varias palabras). Por ejemplo, "zapatillas" sería una palabra clave genérica, mientras que "zapatillas para correr una maratón de montaña", sería una palabra clave long tail. Estos términos suelen tener pocas búsquedas, pero si conseguimos posicionar nuestra web para muchas palabras clave long tail, al final habremos sumado una fuente de tráfico muy relevante.
